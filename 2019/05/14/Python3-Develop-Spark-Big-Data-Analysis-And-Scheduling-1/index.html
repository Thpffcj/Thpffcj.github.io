3. Hadoop

  • 我们使用的hadoop-2.6.0-cdh5.7.0
  • 下载解压，配置环境变量

修改配置文件

  • hadoop-env.sh，这里我并没有做修改，没有写成绝对路径
export JAVA_HOME=${JAVA_HOME}
  • core-site.xml
<configuration>
    <property>
        <name>fs.default.name</name>
        <value>hdfs://127.0.0.1:8020</value>
    </property>


    <property>
        <name>hadoop.tmp.dir</name>
        <value>/Users/thpffcj/Public/software/tmp</value>
    </property>
</configuration>
  • hdfs-site.xml，这里我感觉和上面配置重复了，以前学的时候上面配置了hadoop.tmp.dir，底下没有配置dfs.namenode.name.dir和dfs.namenode.data.dir，以前配置参考分布式文件系统HDFS
<configuration>
   <property>
        <name>dfs.namenode.name.dir</name>
        <value>/Users/thpffcj/Public/software/tmp/dfs/name</value>
    </property>

   <property>
        <name>dfs.namenode.data.dir</name>
        <value>/Users/thpffcj/Public/software/tmp/dfs/data</value>
    </property>

    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
</configuration>
  • mapred-site.xml
<configuration>
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
</configuration>
  • yarn-site.xml 
<configuration>

<!-- Site specific YARN configuration properties -->
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
</configuration>
  • 启动HDFS
Thpffcj:sbin thpffcj$ ./start-dfs.sh

Thpffcj:sbin thpffcj$ jps
20721 SecondaryNameNode
20789 Jps
20536 NameNode
20618 DataNode

6. Spark

spark 安装

  • 在官网下载Spark源码
Thpffcj:software thpffcj$ tar -zxvf spark-2.3.3.tgz

  • Building Spark
Thpffcj:spark-2.3.3 thpffcj$ ./dev/make-distribution.sh --name 2.6.0-cdh5.7.0 --tgz -Pyarn -Phadoop-2.6 -Phive -Phive-thriftserver -Dhadoop.version=2.6.0-cdh5.7.0
  • 我们会在目录下得到spark-2.3.3-bin-2.6.0-cdh5.7.0.tgz，解压
Thpffcj:software thpffcj$ tar -zxvf spark-2.3.3-bin-2.6.0-cdh5.7.0.tgz
