循环神经网络

1. 循环神经网络

1. 为什么需要循环神经网络 - 序列式问题

  • 1对多：图片生成描述
  • 多对1：文本分类(文本情感分析)
  • 多对多：encoding-decoding，机器翻译
  • 实时多对多：视频解说

2. 循环神经网络

  • 维护一个状态作为下一步的额外输入
  • 每一步使用同样的激活函数和参数

  • 反向传播
    • Tanh输出在-1和1之间
    • 梯度消失
    • 较远的步骤梯度贡献很小
    • 切换其他激活函数后，可能也会导致爆炸

3. 多层网络与双向网络

多层网络

  • 底层输出作为高层输入
  • 同层之间依旧递归
  • 增加网络拟合能力
  • 一般隐层维数递增
    • 64 - 128 - 256

双向网络

  • 另一路以未来状态为输入
  • 两个状态拼接后进入输出层
  • 进一步提高表达能力
  • 无法实时输出结果

4. 长短期记忆网络(LSTM)

为什么需要LSTM

  • 普通RNN的信息不能长久传播(存在于理论上)
  • 引入选择性机制
    • 选择性输出
    • 选择性输入
    • 选择性遗忘
  • 选择性 -> 门
    • Sigmoid函数：[0， 1]

  • 遗忘门
    • 新的一句有新的主语，就应该把之前的主语忘掉
  • 传入门
    • 是不是要把主语的性别信息添加进来
  • 输出门
    • 动词该用单数形式还是复数形式
  • 当前状态
    • 经过遗忘门的上一状态
    • 经过输入门的输入状态


2. 文本分类模型

1. LSTM文本分类

  • 使用LSTM的最后一个状态
    • 输入embedding
    • 输出全连接层
  • 双向LSTM解决信息瓶颈

2. HAN文本分类

3. 基于CNN的文本分类模型

  • 需要每个句子截取为同样长度，可适用于短文本分类

  • 单通道一维卷积
  • 多通道一维卷积
  • 多个卷积核会有多个输出

CNN文本分类

  • 一维卷积
    • 应用在时间维度上
    • Embedding长度就是通道数目
    • 多种层次的卷积核
  • 池化
    • 在时间层次上pooling
  • 全连接

4. CNN vs RNN

  • CNN不能完美的解决序列式问题
  • CNN卷积相当于N-gram，LSTM提取更长的依赖
  • Pre-train的embedding
  • 双向RNN会增强效果
  • CNN模型并行程度高，更快
  • embedding模型压缩

5. R-CNN文本分类

  • 双向RNN提取特征
  • CNN进一步抽取
  • Max-pooling

  • 全连接层

    6. Embedding压缩

  • Embedding层次参数过大

    • 无法实用
    • 过拟合
