二分搜索树

我们使用数据结构的核心是为了解决问题，而不是为了使用数据结构而使用数据结构。二叉搜索树能够高效解决计算机中非常常用的查找问题。

1. 二分查找法

在介绍二分搜索树之前，我们先了解一个查找问题的基础二分查找法。

  • 对于有序数列，才能使用二分查找法(排序的作用)
  • 二分查找法的思想在1946年提出。
  • 第一个没有bug的二分查找法在1962年才出现。

最重要的地方就是定义清楚每个变量的定义
非递归的二分查找算法

/**
 * Created by Thpffcj on 2018/3/12.
 * 非递归的二分查找算法
 */
public class BinarySearch {

    // 我们的算法类不允许产生任何实例
    private BinarySearch() {
    }

    // 二分查找法,在有序数组arr中,查找target
    // 如果找到target,返回相应的索引index
    // 如果没有找到target,返回-1
    public static int find(Comparable[] arr, Comparable target) {

        // 在arr[l...r]之中查找target
        int l = 0;
        int r = arr.length - 1;
        while (l <= r) {

            // int mid = (l + r)/2;
            // 防止极端情况下的整形溢出，使用下面的逻辑求出mid
            int mid = l + (r - l) / 2;

            if (arr[mid].compareTo(target) == 0) {
                return mid;
            }

            if (arr[mid].compareTo(target) > 0) {
                r = mid - 1;
            } else {
                l = mid + 1;
            }
        }

        return -1;
    }
}

递归的二分查找算法

/**
 * Created by Thpffcj on 2018/3/12.
 * 递归的二分查找算法
 */
public class BinarySearch2 {

    // 我们的算法类不允许产生任何实例
    private BinarySearch2() {
    }

    private static int find(Comparable[] arr, int l, int r, Comparable target) {

        if (l > r) {
            return -1;
        }

        //int mid = (l+r)/2;
        // 防止极端情况下的整形溢出，使用下面的逻辑求出mid
        int mid = l + (r - l) / 2;

        if (arr[mid].compareTo(target) == 0) {
            return mid;
        } else if (arr[mid].compareTo(target) > 0) {
            return find(arr, l, mid - 1, target);
        } else {
            return find(arr, mid + 1, r, target);
        }
    }

    // 二分查找法,在有序数组arr中,查找target
    // 如果找到target,返回相应的索引index
    // 如果没有找到target,返回-1
    public static int find(Comparable[] arr, Comparable target) {

        return find(arr, 0, arr.length - 1, target);
    }
}
  • 递归实现通常思维起来更容易。
  • 递归在性能上会略差。
