Scala 基础

1. Scala的语言基础

1. 函数式编程

  • In computer science, functional programming is a programming paradigm—a style of building the structure and elements of computer programs—that treats computation as the evaluation of mathematical functions and avoids changing-state and mutable data.
  • 重要概念
    • 纯函数，或函数的纯粹性，没有副作用
      • 副作用是状态的变化：修改全局变量，抛出异常，IO读写，调用有副作用的函数
    • 引用透明：对于相同的输入，总是到相同的输出
      • 如果f(x)的参数x和函数体都是引用透明的，那么函数f是纯函数
      • 不变性：为了获得引用透明性，任何值都不能变化
    • 函数是一等公民：一切都是计算，函数式编程中只有表达式，变量，函数都是表达式
      • 高阶函数
      • 闭包
    • 表达式求值策略：严格求值和非严格求值
      • Call By Value vs Call By Name
      • 惰性求值
    • 递归函数
      • 递归实现循环
      • 尾递归

2. 函数式编程优点

  • 生产效率高
    • 同样功能的程序，Lisp代码的长度可能是C代码的1/7~1/10
  • 易于推理
  • 并行编程
    • 多核计算，云计算

3. 函数式编程的崛起

  • 摩尔定律
  • 多核处理器
  • 云计算时代


2. Scala基础语法

1. 变量

三种变量修饰符

  • var 定义 immutable variable
  • var 定义 mutable variable
  • lazy val
  • 可以不显示指明变量的类型，因为Scala会自动进行类型推倒
scala> val x = 10
x: Int = 10

scala> val y:Int = 20
y: Int = 20

scala> x + x
res0: Int = 20

scala> res0
res1: Int = 20

scala> res0*res1
res2: Int = 400

scala> val z = res0 + res1 + res2
z: Int = 440
  • 这个时候如果我们给z赋值，就会报错
scala> z = 100
<console>:12: error: reassignment to val
       z = 100
  • 如果要修改变量的值，需要用var修饰变量名
scala> var a = 200
a: Int = 200

scala> a = 300
a: Int = 300
  • 我们看到声明lazy是不会立即求值的
scala> val d = 20
d: Int = 20

scala> val e = 30
e: Int = 30

scala> lazy val f = d * e
f: Int = <lazy>
  • 当它被用到的时候就会求解
scala> f * 10
res3: Int = 6000

scala> f
res4: Int = 600
  • 什么时候我们会定义lazy val：我们定义的变量在后续的程序中可能不会被用到

2. 数据类型

  • Any是所有Scala类型的父类
  • 总体上，Scala的对象可分为两个类型：
    • 引用类型(Reference Types)：继承自AnyRef
    • 值类型(Value Types)：继承自AnyVal
scala> val a:Byte = 10
a: Byte = 10

scala> val b:Short = 20
b: Short = 20

scala> val c:Int = 30
c: Int = 30

scala> val d:Long = 40
d: Long = 40

scala> val e:Float = 50
e: Float = 50.0

scala> val f:Double = 60.98765
f: Double = 60.98765
  • 用高精度赋值低精度就会报错
scala> val x:Long = b
x: Long = 20

scala> val y:Byte = x
<console>:12: error: type mismatch;
 found   : Long
 required: Byte
       val y:Byte = x
                    ^
  • 接着看一些其他类型
scala> val m = true
m: Boolean = true

scala> val n = false
n: Boolean = false

scala> val q = 'X'
q: Char = X

scala> val u:Unit = ()
u: Unit = ()

scala> val p = ()
p: Unit = ()

scala> def foo() = throw new Exception("error occurred")
foo: ()Nothing
  • String
    • 构建于Java的String之上
    • 新增了字符串插值的特性
scala> val name = "Thpffcj"
name: String = Thpffcj

scala> s"my name is ${name}"
res0: String = my name is Thpffcj

3. 函数与代码块

  • Block也是一个表达式，其最终的求得的值是最后一个表达式的值
object func_examples {
  def hello(name:String):String = {
    s"Hello, ${name}"
  }

  hello("Thpffcj")

  def hello2(name:String) = {
    s"Hello, ${name}"
  }

  hello2("Thpffcj")

  def add(x:Int, y:Int) = x + y

  add(1, 2);
}

4. if与for

  • if (logical_exp) valA else valB
object for_example {
  val l = List("alice", "bob", "cathy")

  for (
    s <- l
  ) println(s)

  for {
    s <- l
    if (s.length > 3)
  } println(s)

  val result_for = for {
    s <- l
    s1 = s.toUpperCase()
    if (s1 != "")
  } yield (s1)
}

5. try与match表达式

object try_example {
  val result_try = try {
    Integer.parseInt("dog")
  } catch {
    case _ => 0
  } finally {
    println("always be printed")
  }

  val code = 3
  val result_match = code match {
    case 1 => "one"
    case 2 => "two"
    case _ => "others"
  }
}

6. 求值策略

  • Scala里有两种求值策略
    • Call By Value - 对函数实参求值，且仅求值一次
    • Call By Name - 函数实参每次在函数体内备用到时都会求值
  • Scala通常使用Call By Value
    • 如果函数形参类型以 => 开头，那么就会使用Call By Name


3. 高阶函数

1. 函数是第一等公民

  • 把函数作为实参传递给另外一个函数
  • 把函数作为返回值
  • 把函数赋值给变量
  • 把函数存储在数据结构里
  • 在Scala中，函数就像普通变量一样，同样也具有函数的类型
